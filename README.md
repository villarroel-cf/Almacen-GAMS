# ALMACEN G.A.M.S. API

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
4. Cambiar las variables de entorno
5. Levantar la base de datos
```docker-compose up``` o ```docker-compose up -d```
6. Levantar la aplicacion: ```yarn start:dev```
